Staff Panel APIs

This document provides a comprehensive overview of the APIs designed for the Staff Application for CHITTI.

Course Content Management​

These endpoints allow staff to perform CRUD operations on course resources such as Videos, Notes, Cheatsheets, and Important Questions for specific topics and units within a course.

Get Videos​

Method: GET
Path: /:course_id/units/:unit_id/videos
Description: Fetches all videos for a specific unit in a course.
Path Params:
course_id: The ID of the course.
unit_id: The ID of the unit.
Request Header:

{
  "X-App-Version": "X.Y.Z",
  "X-Device-Id": "string",
  "Authorization": "Bearer ey..."
}

Response:

  • Success200
  • Unexpected Error500
{
  "message": "Videos fetched successfully.",
  "status": true,
  "data": [
    {
      "video_id": "string",
      "course_id": "string",
      "unit_id": "string",
      "topic_id": "string",
      "name": "string",
      "url": "string",
      "thumbnail": "string"
    }
  ]
}
{
  "message": "Couldn't fetch videos.",
  "status": false,
  "error_code": "videos/internal-server-error"
}

Create Notes​

Method: POST
Path: /:course_id/units/:unit_id/topics/:topic_id/notes
Description: Creates a new notes resource for a topic.
Path Params:
course_id: The ID of the course.
unit_id: The ID of the unit.
topic_id: The ID of the topic.
Request Body:

{
  "name": "string",
  "url": "string"
}

Response:

  • Success200
  • Invalid Request400
  • Unexpected Error500
{
  "message": "Notes created successfully.",
  "status": true,
  "data": {
    "notes_id": "string",
    "course_id": "string",
    "unit_id": "string",
    "topic_id": "string",
    "name": "string",
    "url": "string"
  }
}
{
  "message": "Malformed payload. Required: notes_id, name, url",
  "status": false,
  "error_code": "payload/misformed-payload"
}
{
  "message": "Couldn't create notes.",
  "status": false,
  "error_code": "notes/internal-server-error"
}
